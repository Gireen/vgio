# Hxshader

## Header

| Offset | Size | Type |    Description     |              Notes              |
|--------|------|------|--------------------|---------------------------------|
| 0x00   | 4    | int  | name_size          |  Length of shader name.         |
| 0x04   | 4    | int  | vertex_shader_size |  Length of the vertex shader.   |
| 0x08   | 4    | int  | frag_shader_size   |  Length of the fragment shader. |
